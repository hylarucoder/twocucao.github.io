Mac 上开发者必备的软件

发表于 2016-11-23 | 分类于 PC/MAC 使用心得技巧
| 字数统计 | 阅读时长

前言

好的工匠懂得挑选合适的工具。

做软件行业之间长了，见多了各种操作系统孰优孰劣 / 编程语言哪家强的论战，也就渐渐懂得了这个异常朴素的道理。也懒得去争论。有争论的时间，不如好好的编写代码，多看些技术书籍。以及熟悉自己的工具。

如果说，你现在问我到底是哪个 OS 好，我只能说：

好的工匠懂得挑选合适的工具。而不是炫耀自己的工具。

大约一年前入手了一个 Macbook Pro，一年后，更加喜欢 Mac 上面的软件。

目前主要工作范围集中在：

  • Python 数据爬虫
  • Django Web 开发

现在介绍我使用的一些工具和配置。

1. 日常工具

基本工具代表这些工具是每一台 Mac 都值得一试的工具。

  • 启动器 Alfred 3
  • 浏览器 Chrome Firefox
  • 输入法 搜狗输入法
  • 系统清理 AppCleaner, CleanMyMac
  • 系统增强 Caffeine,PopClip
  • 手机管理工具 HandShaker , AirDroid
  • 邮件客户端 Airmail 2
  • 录屏截图 Annotate, Camtasia 2, Snagit
  • 下载工具 Aria2GUI , 迅雷，百度网盘
  • 影音处理 Adobe PhotoShop CC , Adobe PhotoShop , Adobe LightingRoom , Final Cut Pro
  • 影音浏览 MPlayerX, QuickTime , IQIYI
  • 远程协助 TeamViewer
  • 办公软件 Wiznote , PDF Expert , Office（虚拟机内部）,XMIND

2. 开发工具

虽然用 Mac 的程序员一般情况下都是键盘命令行流，但我更加倾向于在尽量减少记忆负担的情况下，尽量使用键盘完成工作。

起始开发环境的配置建议参考，https://github.com/donnemartin/dev-setup

  • GIS 相关 QGIS , PostgreSQL + PostGIS
  • IDE 选择 JetBrain 家的软件 PyCharm, IntellijIDEA
  • 编辑器 MacVim （主力）, 配合 K-VIM 作为日常编写文字的利器。
  • 数据库 MySQL , PostgreSQL（主力） , Redis , MongoDB
  • 数据管理 Navicat,Datagrip,RoboMongo,rdm
  • 文档查看 Dash
  • 网络工具 SS QT 不解释
  • 网络抓包 Charles, Wireshark, Chrome F12

命令行工具

  • linux commom tools
  • mac 独有命令行
    • open
    • pbcopy
    • pbpaste
    • screencapture
    • launchctl
    • mdfind（还是 linux 的 find 好用）
    • sip （还是比较推荐 imagemagic)
  • imagemagic
  • nmap
  • tesseract
  • mpv
